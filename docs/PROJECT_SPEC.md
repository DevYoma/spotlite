# PROJECT_SPEC.md

## Vision

A platform that helps organizations collect information through customizable forms and automatically generate branded graphics from those submissions.

## Problem

Today, organizations manually:

1. Collect information through forms
2. Organize responses
3. Create graphics one by one
4. Download and distribute them

This process is repetitive and time-consuming.

## Solution

Users create forms, collect submissions, create templates, and generate graphics automatically.

## Target Users

* Student associations
* Graduating classes
* Schools
* Companies
* Churches
* Communities
* Event organizers

## MVP Features

* Authentication
* Projects
* Dynamic Form Builder
* Public Forms
* Form Submissions
* Template Creation
* Graphic Generation
* Image Downloads

## Success Criteria

A user can:

1. Create a project
2. Create a form
3. Collect submissions
4. Create a template
5. Generate a graphic
6. Download the generated graphic

## Out of Scope

* WhatsApp Automation
* Scheduling
* AI Templates
* Team Collaboration
* Analytics
* Payments
* Template Marketplace
* Bulk Generation
* Public APIs

These will be evaluated after MVP validation.
